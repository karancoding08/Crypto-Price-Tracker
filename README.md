# Crypto Price Tracker

## 📌 Project Description
This project is a simple Python application that fetches real-time cryptocurrency prices using the CoinGecko API.

Users can enter a cryptocurrency name like Bitcoin, Ethereum, or Dogecoin and get the current price in USD.

---

## 🚀 Features
- Real-time crypto price fetching
- User input support
- JSON data parsing
- API integration using requests
- Error handling using try-except

---

## 🛠 Technologies Used
- Python
- Requests module
- CoinGecko API
- JSON handling
