# Crypto Price Tracker

## 📌 Project Description

This project is a simple Python application that fetches real-time cryptocurrency prices using the CoinGecko API.

Users can enter a cryptocurrency name like Bitcoin, Ethereum, or Dogecoin and get the current price in USD.

---

## 🚀 Features

* Real-time crypto price fetching
* User input support
* JSON data parsing
* API integration using requests
* Error handling using try-except

---

## 🛠 Technologies Used

* Python
* Requests module
* CoinGecko API
* JSON handling

---

## ▶️ How to Run

1. Install the required library:

pip install requests

2. Run the program:

python main.py

3. Enter a coin name such as:

bitcoin

ethereum

dogecoin

---

## 📸 Output Example

The program displays the current price of the selected cryptocurrency in USD.
